# dotfiles
To ease the pain.

## tmux
Clone TPM before running tmux.
```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
Reload tmux if it is already running.
```bash
tmux source ~/.tmux.conf
```
Press `prefix` + `I` to install plugins.
